# 📝 Gestor de Tareas GUI con Atajos de Teclado

Aplicación de escritorio desarrollada en Python utilizando Tkinter que permite gestionar tareas pendientes mediante interfaz gráfica y atajos de teclado.

---

## 🎯 Objetivo

Desarrollar una aplicación GUI que permita:
- Añadir tareas
- Marcar tareas como completadas
- Eliminar tareas
- Usar atajos de teclado para mejorar la usabilidad

---

## 🖥️ Tecnologías utilizadas

- Python 3
- Tkinter

---

## 🚀 Funcionalidades

### 🖱️ Interfaz gráfica
- Campo de entrada para nuevas tareas
- Lista de tareas
- Botones:
  - Añadir
  - Completar
  - Eliminar

### ⌨️ Atajos de teclado

| Acción | Tecla |
|------|------|
| Añadir tarea | Enter |
| Completar tarea | C |
| Eliminar tarea | Delete / D |
| Salir | Escape |

---

## 🎨 Feedback visual

- ✔ Tareas completadas → color gris
- • Tareas pendientes → color negro
